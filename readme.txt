The files in this repository are:

DATA - All the data acquired from the prototype used to obtain the thesis results;
RNN-BPTT - Recurrent neural network with two hidden layers trained by the back-propagation through time algorithm [MATLAB];
RNN-EKF - Recurrent neural network with two hidden layers trained by the extended Kalman filter algorithm [MATLAB];
B-LSTM - Long short-term memory network [Python and Tensorflow].
